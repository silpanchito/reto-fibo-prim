Fibonacci
en la Naturaleza

¿Cuál es el problema real?

En la naturaleza, la serie de Fibonacci aparece en los patrones de crecimiento de plantas, flores y ramas. Esta página modela cuántas ramas nuevas aparecen en cada nivel de una planta, siguiendo la progresión matemática que ocurre en la realidad.

🌸

Pétalos de flores

La mayoría de flores tienen 3, 5, 8 o 13 pétalos — siempre números de Fibonacci. Es la forma más eficiente de empacar pétalos alrededor de un centro.

🌿

Ramificación de plantas

Cada rama que nace de un tronco sigue el patrón Fibonacci: 1 → 1 → 2 → 3 → 5 → 8… maximizando la luz captada por la planta.

🐚

Espiral áurea

La razón entre términos consecutivos se aproxima a φ = 1.618 (número áureo). Esta proporción define la espiral visible en caracoles y galaxias.

¿Cómo funciona el algoritmo?

01

La Serie de Fibonacci

Empieza con 0 y 1. Cada número siguiente es la suma de los dos anteriores:
0 → 1 → 1 → 2 → 3 → 5 → 8 → 13 → 21…

02

Aplicación en Plantas

Cada nivel de crecimiento produce tantas ramas nuevas como indica el término correspondiente de la serie. El modelo refleja el crecimiento real.

03

El Algoritmo en JavaScript

Solo tres variables (a, b, c) y un ciclo for. Sin arreglos: sumas sucesivas. Resultado mostrado con getElementById().

0 + 1 → 1 → 2 → 3 → 5 → 8 → 13 → …
ALGORITMO FIBONACCI — SIN ARREGLOS JavaScript 
// Solo variables simples, sin arrays
let a = 0;
let b = 1;
let c;

for (let i = 3; i <= niveles; i++) {
    c = a + b;   // suma de los dos anteriores
    a = b;
    b = c;
}

// Mostrar en la página con getElementById
document.getElementById("resultadoFibonacci").innerHTML = c;

Conclusión

Fibonacci no es solo matemática abstracta: es el lenguaje del crecimiento. Comprender este patrón permite modelar procesos naturales, diseñar estructuras eficientes y predecir comportamientos en biología, arquitectura y tecnología.

Números Primos
en la Música

¿Cuál es el problema real?

Esta página detecta números primos dentro de un rango de frecuencias ingresado por el usuario y los relaciona con notas musicales reales. El algoritmo recorre el rango con un ciclo for y cuenta divisores con el operador módulo para verificar si cada número es primo.

🎼

Compases asimétricos

Un compás de 5/4 o 7/8 (números primos) crea ritmos que no pueden dividirse uniformemente, generando tensión y originalidad únicas en la música.

🎹

Frecuencias especiales

Las frecuencias primas (Hz) no forman armonías perfectas con otras, creando disonancias interesantes usadas en música contemporánea y experimental.

🔐

Criptografía musical

Compositores como Messiaen y Ligeti usaron secuencias de primos para crear melodías impredecibles — igual que en la encriptación de datos.

Matemáticas y música en imágenes

¿Cómo funciona el algoritmo?

01

¿Qué es un número primo?

Un número es primo si solo tiene dos divisores: el 1 y él mismo. El 7 es primo. El 6 no, porque también lo dividen 2 y 3.

02

Cómo lo detectamos

Recorremos con for desde 1 hasta el número. Contamos cuántos lo dividen exactamente (% == 0). Si el contador llega a 2: es primo.

03

Aplicación al rango

El usuario ingresa un rango de frecuencias (Hz). El programa aplica la verificación a cada número y muestra los primos con su nota musical más cercana.

VERIFICACIÓN DE NÚMERO PRIMO — SIN ARREGLOS JavaScript 
var numero = parseInt(document.getElementById("numero").value);
var contador = 0;

for (var i = 1; i <= numero; i++) {
    if (numero % i == 0) {
        contador++;
    }
}

if (contador == 2) {
    document.getElementById("resultado").innerHTML = "Es primo";
} else {
    document.getElementById("resultado").innerHTML = "No es primo";
}

Conclusión

Los números primos en la música crean ritmos únicos e irrepetibles. Compositores como Messiaen y Ligeti los usaron para romper con la monotonía. Entender la primalidad ayuda a diseñar ritmos originales y — fuera de la música — a proteger información en criptografía.

Fibonacci & Primos
juntos

¿Cuál es el problema real?

Dentro de la serie de Fibonacci existen números que también son primos: 2, 3, 5, 13, 89, 233… Son extremadamente raros. Esta herramienta genera la serie Fibonacci, aplica la verificación de primalidad a cada término y los relaciona con notas musicales si están en el rango audible.

01

Generar Fibonacci

Se genera la serie usando solo variables a, b, c y un ciclo for, sin arreglos.

02

Verificar primalidad

Cada término se pasa por el algoritmo de contador con módulo. Si el contador es exactamente 2, es primo.

03

Nota musical

Si el valor cae en el rango audible (16 – 4186 Hz), se muestra la nota musical que le corresponde.
